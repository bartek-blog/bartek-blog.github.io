## Hello World 

Create `HelloWorld.scala` file with the folloeing
``` scala
object CubeCalculator extends App {
    println("Hello World")
}
```
Compile it by 

``` shell
scalac HelloWorld.scala
```
Run it with:

``` shell
scala HelloWorld
```
